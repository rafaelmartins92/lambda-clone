# Lambda.ai Clone 🚀

A pixel-perfect reconstruction of the Lambda.ai website built with React, showcasing modern web development practices and attention to detail in UI implementation.

## 🌐 Live Demo

**[View Live Demo](https://lambda-clone.vercel.app/)**

## 📝 Project Overview

This project is a front-end challenge that demonstrates the ability to recreate complex, production-grade interfaces with precision. The clone faithfully reproduces the Lambda.ai website's design, animations, and user experience.

## ✨ Features

- **Pixel-Perfect Design** - Accurate recreation of the original Lambda.ai interface
- **Responsive Layout** - Fully responsive design that works across all devices
- **Modern Animations** - Smooth transitions and interactive elements
- **Component Architecture** - Modular, reusable React components
- **Performance Optimized** - Fast loading times and efficient rendering
- **Clean Code** - Well-structured and maintainable codebase

## 🛠️ Technologies Used

- **React 18.3** - Modern React with hooks
- **Vite 5.4** - Next-generation frontend tooling
- **Tailwind CSS 3.4** - Utility-first CSS framework
- **CSS Modules** - Scoped styling for components
- **ESLint** - Code quality and consistency
- **PostCSS** - CSS transformations

## 🚀 Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/rafaelmartins92/lambda-clone.git
cd lambda-clone
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## 📜 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint for code quality
